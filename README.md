# About

This repo includes inspo experiments like:

- **Theme pickers** – switch between dark, soft, and light modes with custom color palettes.
- **Interactive maps** – simple, responsive maps to showcase locations and spatial data.
- **UI components** – buttons, cards, and other reusable elements for web projects.

The goal is to explore different concepts, improve development workflow, and gradually build a collection of useful components.

## Demo

A live demo of each components can be found in jmbstudio.dev or by running the project locally.
